About

My Name is Mike Kidushim and I am a full stack web developer. I love solving complex problems with clean, elegant solutions. I'm also proficient in web programming languages such as HTML5, CSS3, JavaScript, PHP, and SQL. In my free time, I enjoy continually building on my skills by studying advancements in industry trends by going to meetups, reading and just messing around with new ideas.

I have experience with all phases of the web development life cycle: establishing client and project requirements; converting mockups to code; and finally, deploying fully functional websites. Most things I build are done so using an MVC structure with php, javascript/JQuery/Angular/React to create intuitive and clean UI/UX that communicates with the API I create in php. I also manage LAMP servers running on windows and linux operating systems, keeping them running at peek performance. If you are interested in getting started on a new website you can contact me using the form below.
